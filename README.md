# Avenza - Full Stack Ecommerce Website

A modern full-stack ecommerce web application built with **React, Tailwind CSS, Node.js, Express.js, and MongoDB**.  
Avenza provides a smooth shopping experience with authentication, cart management, wishlist, orders, reviews, Stripe checkout, and admin product/order management.

---

## Live Demo

- **Frontend:** https://myavenza.onrender.com
---

## Features

### User Side
- User signup and login
- Google authentication
- Forgot password / reset password with OTP
- Browse products by category
- Search products
- Add to cart
- Wishlist / favorite products
- Buy now functionality
- Stripe checkout integration
- Order placement and order tracking
- Product reviews and ratings
- Responsive design for mobile, tablet, and desktop
- Dark mode / light mode UI

### Frontend
- React.js
- Tailwind CSS
- React Router DOM
- Axios
- Framer Motion
- Lottie React
- React Icons

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Cookie Parser
- Multer
- Stripe
